# Инструкция по загрузке фотографий

## Структура папок

Загружайте фотографии в соответствующие папки:

- **green-bedroom/** - Зеленая спальня
- **white-bedroom/** - Белая спальня
- **secret-room/** - Секретная комната
- **sauna/** - Сауна
- **living-kitchen/** - Гостиная и кухня
- **terrace/** - Терраса

## Формат изображений

- **Рекомендуемый формат:** JPG или WebP
- **Рекомендуемый размер:** 1200x900 px (соотношение 4:3)
- **Называйте файлы:** 1.jpg, 2.jpg, 3.jpg и т.д.

## Пример

```
rooms/
├── green-bedroom/
│   ├── 1.jpg
│   ├── 2.jpg
│   └── 3.jpg
├── white-bedroom/
│   ├── 1.jpg
│   └── 2.jpg
└── ...
```

## После загрузки фотографий

Нужно обновить файл `src/data/gallery-manifest.json`, заменив ссылки на Unsplash на локальные:

Вместо:
```json
"src": "https://images.unsplash.com/photo-123..."
```

Используйте:
```json
"src": "/images/rooms/green-bedroom/1.jpg"
```

## Автоматическое обновление

После замены ссылок в gallery-manifest.json, dev-сервер автоматически обновит галерею.
